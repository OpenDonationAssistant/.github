## **OpenDonationAssistant** - an open-source platform for streamers
Website: [https://oda.digital/](https://oda.digital/)

Components:
- oda-widget-page - all widgets and dashboard. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-widget-page/)
- oda-payment-page - donation page. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-payment-page/)
- oda-config-service - service with streamer's configs for donation-page. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-config-service/)
- oda-libraries - shared libs for java services. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-libraries/)
- oda-website - website for OpenDonationAssistant. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-website/)
- oda-history-service - service which handles events history - donations, subscriptions, etc. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-history-service/)
- oda-payment-service - service which processes donations. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-payment-service/)
- oda-media-service - service which handles media requests by donaters, chatters and streamers. Also it stores playlists and some other metadata. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-media-service)
- oda-font-service - service which stores and updates links to fonts. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-font-service/)
- oda-automation-service - service which stores and processes automation rules. [Wiki](https://deepwiki.com/OpenDonationAssistant/oda-automation-service/)

Code rules, usable by ai agents: [gist](https://gist.github.com/stCarolas/aa0ba93d72b6e7c138ca6cef81d529b1)
